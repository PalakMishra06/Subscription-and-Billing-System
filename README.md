# Subscription and Billing System

![Project Badge](https://img.shields.io/badge/status-active-brightgreen)

A full-stack web application to manage user subscriptions and billing. Users can view plans, subscribe, manage payments, and track active subscriptions.

---

## Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Java Spring Boot
- **Database:** MySQL
- **Architecture:** RESTful APIs, layered (Controller → Service → Repository)

---

## Features

- User registration and login
- View and subscribe to plans
- Manage payments and subscriptions
- Admin can manage plans and users
- Responsive frontend UI
